## 📖 Keyboard shortcuts

Learning a few shortcuts speeds up your daily work:

*   L: Starts the Line tool.
*   C: Starts the Circle tool.
*   PL: Starts the Polyline tool.
*   TR: Trims lines that intersect with other objects.
*   EX: Extends lines to a specified boundary.
*   M: Moves objects from one coordinate to another.
*   CO: Copies selected objects.
*   RO: Rotates objects around a base point.
